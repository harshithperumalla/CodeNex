# SparkLearn Hub 🚀

A modern coding, aptitude, and learning platform built with React.

## Features

- 💻 Coding Arena with interactive problems
- 🧠 Aptitude quizzes with concept-based learning
- 🗣 Communication & Interview preparation
- 🎮 Games for logic and practice
- 👨‍🏫 Mentor system and meetings
- 📊 Progress tracking and analytics

## Tech Stack

- React + Tailwind CSS
- Vite
- Modern UI/UX design

## Getting Started

```bash
npm install
npm run dev
